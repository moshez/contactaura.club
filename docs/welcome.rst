Welcome to My RPG Materials
============================

This is a demonstration file showing how RPG materials are organized in this repository.

What You'll Find Here
---------------------

This site is dedicated to hosting various RPG (Role-Playing Game) materials, including:

* **House Rules**: Custom rules and modifications
* **Adventures**: Original adventures and scenarios
* **NPCs**: Non-player characters with backgrounds and stats
* **Locations**: Detailed descriptions of places and settings
* **Items**: Custom magic items, equipment, and artifacts
* **Reference Materials**: Quick reference guides and tables

How to Use This Site
--------------------

All materials are organized as separate documents in the ``docs/`` directory.
Simply add new ``.rst`` (reStructuredText) files to the ``docs/`` folder,
and they will automatically appear in the table of contents.

Adding New Materials
--------------------

To add new RPG materials:

1. Create a new ``.rst`` file in the ``docs/`` directory
2. Write your content using reStructuredText formatting
3. Rebuild the site using the ``build.sh`` script
4. Your new document will automatically appear in the navigation

Example Document Structure
---------------------------

Here's a sample structure for an adventure document::

    Adventure Title
    ===============

    Synopsis
    --------
    Brief description of the adventure.

    Background
    ----------
    History and context.

    Encounters
    ----------
    Detailed encounter descriptions.

Getting Started
---------------

Browse the navigation menu to explore available materials. Each document
is self-contained and can be referenced independently during game sessions.

Happy Gaming!
-------------

May your dice roll high and your adventures be memorable!
